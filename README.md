# London Transit SQL Analysis (TfL)

## Overview
Analyzed Transport for London Rolling Origin and Destination Survey (RODS) data
using SQL to identify ridership patterns, peak travel periods, and trip purposes.

## Skills & Tools
- SQL (SUM, GROUP BY, ORDER BY, WHERE, HAVING)
- Aggregation and multi-column grouping
- Analytical reasoning for transit planning

## Key Insights
- Identified peak travel periods and highest-volume zones
- Analyzed journey purposes by time of day and location
- Generated data-driven recommendations for transit optimization

## Repository Contents
- `queries.sql` – SQL queries used for analysis
- `milestone.pdf` – Completed project submission with explanations and findings
